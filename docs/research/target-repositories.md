# 目标仓库基线

## 首轮 POC

### RuoYi-Vue

- 仓库：`yangzongzhuan/RuoYi-Vue`
- 技术栈：Java / Spring Boot / Maven / Vue
- 角色：Java 企业后台系统基线
- 选择理由：模块清晰、常见 B 端架构、体量适中，适合首轮 POC

### eShopOnWeb

- 仓库：`dotnet-architecture/eShopOnWeb`
- 技术栈：.NET / ASP.NET Core / EF Core
- 角色：非 Java 跨技术栈基线
- 选择理由：架构清晰、项目结构明确、有测试覆盖、运行成本低

## 第二阶段候选

### Apache Fineract

- 仓库：`apache/fineract`
- 技术栈：Java / Spring Boot / Gradle / PostgreSQL
- 角色：复杂业务系统验证候选
- 选择理由：真实金融系统、模块图谱庞大、具有强企业 / 业务代表性
- 首轮定位：仅作为静态基线参考，不作为完整构建目标
